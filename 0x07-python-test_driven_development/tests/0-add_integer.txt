How to use ``0-add_integer.py`` module
======================================

The module defines a integer addition function ``add_integer(a, b=98)``.
-----------------------------------------------------------------------

``add_integer(...) `` returns the result of the addition of two integers
, ewuivalent to ``+`` operateur.

	>>> add_integer = __import__('0-add_integer').add_integer
	>>> add_integer(2, 5)
	7
