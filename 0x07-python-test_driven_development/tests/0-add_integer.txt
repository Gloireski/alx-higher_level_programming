How to use ``0-add_integer.py`` module
======================================

The module defines a integer addition function ``add_integer(a, b=98)``.
-----------------------------------------------------------------------

``add_integer(...) `` returns the result of the addition of two integers
, ewuivalent to ``+`` operateur.

	>>> add_integer = __import__('0-add_integer').add_integer
	>>> add_integer(2, 5)
	7

	>>> add_integer(2, -3)
    	-1
	
	Also works with floating points
	>>> add_integer(2.0, 3.0)
	5

	>>> add_integer(2.9, 0.2)
    	2

	>>> add_integer(-2.9, -0.2)
    	-2

	>>> add_integer(2.3, -3)
    	-1

	>>> add_integer(2)
    	100

Non-Numbers
===========
``add_integer()`` expects that both arguments are either integers or floats.
If either argument is a non-integer and non-float, a TypeError is raised:
	>>> add_integer("hello", 3)
	Traceback (most recent call last):
	TypeError: a must be an integer
